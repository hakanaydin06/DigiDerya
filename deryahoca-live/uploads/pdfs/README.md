# PDF Dosyaları Klasörü

Bu klasöre PDF dosyalarınızı yükleyin. Canlı ders sırasında bu PDF'leri seçip öğrencilerinizle paylaşabilirsiniz.

## Kullanım

1. PDF dosyalarınızı bu klasöre kopyalayın
2. Öğretmen panelinden "PDF Seç" butonuna tıklayın
3. Listeden bir PDF seçin
4. PDF tüm öğrencilerin ekranında senkronize olarak görüntülenecektir

## Öneriler

- PDF dosyalarınızın boyutunu optimize edin (max 10MB önerilir)
- Anlaşılır dosya isimleri kullanın
- Yüksek çözünürlüklü görseller içeren PDF'ler yavaş yüklenebilir
